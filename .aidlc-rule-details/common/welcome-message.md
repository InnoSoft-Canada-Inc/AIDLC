I'll guide you through an **adaptive software development workflow** that tailors itself to your needs.

1. **I analyze your workspace** — new or existing project
2. **I gather requirements** — asking questions if needed
3. **I create an execution plan** — you review and approve
4. **We execute together** — with checkpoints at each stage

- **Simple changes** → minimal stages, fast execution
- **Complex features** → full treatment with all safeguards
- **Bug fixes** → diagnose, fix, verify, done

### Your Control Points:

- Review and approve plans before execution
- Request stages be included or excluded
- Complete audit trail of all decisions
